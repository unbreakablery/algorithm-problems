Swap Two Variables

Do you know 4 methods to swap two variables in Javascript?

  • Using a Temporary Variable
  • Using es6(ES2015) Destructuring assignment
  • Using Arithmetic Operators
  • Using Bitwise XOR operator

Example Input:

  • a = 100, b = 200

Example Output:

  • a = 200, b = 100

          
            //Using temp variable
            temp = a;
            a = b;
            b = temp;

            //Using es6(ES2015) Destructuring assignment
            [a, b] = [b, a];

            //Using Arithmetic Operators
            a = a + b;
            b = a - b;
            a = a - b;

            //Using Bitwise XOR operator
            a = a ^ b
            b = a ^ b
            a = a ^ b
